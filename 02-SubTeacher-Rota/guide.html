代理導師排班

平常請假來了，照這張做就好。有疑問再翻後面的完整說明。

每天最常做的：排代理

  1. 打開工具，上方選日期（預設今天）。
  2. 點一下今天要代理的班級（可點多班，會自動標順序）。
  3. 按 排代理，看誰輪到 → 沒問題按 確認並記錄。

請假是分好幾次來的？

沒關係，一次處理一班就好。先排的人會被鎖住，之後再排別班會自動避開他。

頁面最上面的綠框「今天已排定」，會列出今天排過誰。

要換人？

點該班卡片的「換人 / 更多」，用下拉選單指定，就會標成「手動」。可順便填缺席者、原因。

老師其實沒請假 / 排錯了？

在綠框「今天已排定」那筆按 × 移除；他就會回到輪替、下次照樣輪到。

一定要記得

每次改完，按頁面上方 立即下載備份，存一份檔案。
資料只在這台電腦，不備份、換電腦或清快取就會不見。

偶爾才做的

時機做什麼
開學建資料「老師與班級」：先建班級 → 建老師與序號 → 勾任課班級（人多可用「批次匯入」貼名單）。班名是 701、803 這種的，可按「依班名自動分年級」一鍵分好。
新學年重來「備份與重置」→ 設分界日：次數歸零，舊記錄仍保留可查。
看誰輪幾次「記錄統計」：各班下一位是誰、每人次數、當月統計、可匯出 Excel（CSV）。

請勿沒事別按「全部清除」——那會刪掉全部資料。真的要重來也請先下載備份。

完整功能說明

代理導師排班 · 使用手冊

五個分頁：排代理、老師與班級、請假、記錄統計、備份與重置。

一、它是怎麼決定誰代理的（運作邏輯）

  • 全體參與輪替的老師，開學時一起抽一個序號；導師不抽、不列入（導師不在才需要代理，本人不會被排）。
  • 某班要代理時，候選人＝這個班的任課老師，依序號由小到大。
  • 從候選人中，挑「這個班已代理次數最少」的人；同次數就序號小的先上。這等於「還沒輪過的先上、輪過的先跳，等大家追平再回頭」，會自動長出固定的輪替順序。
  • 輪到的人若當天請假或當天已代理別班，就跳過換下一位；請假／被佔用不算他輪過，次數不變，下次仍優先輪到他。
  • 各班獨立計次：某人代理過甲班，只影響甲班的順序，不影響他在乙班的順位。
  • 同一天多個班要代理：先進系統的班先鎖定人選，同一位老師一天最多代理一班。分時段陸續排也適用。
  • 「次數」＝學年分界日之後的代理記錄數 ＋ 起始次數（見第五節）。

二、排代理

日常主畫面。一進來就是這頁。

  • 日期：萬年曆選擇，預設今天。
  • 今天已排定（綠框）：分次排時的現況，列出當天每一筆「班級 → 代理老師」。若老師後來沒請假或排錯，按該筆的 × 移除，他即回到輪替。
  • 選班級：依年級分組、可折疊。點一下班級即選取，多班會自動標 1、2… 的需求順序；同年級內可用 ↑ ↓ 微調順序。
  • 排代理：產生建議。只試排，先不記錄：勾了之後只看結果、不會寫入（適合「想先看看今天會是誰」）。
  • 結果卡片：每班一張，大字顯示代理老師，標「自動輪到」或「手動」。展開「換人 / 更多」可：指定代理老師、填缺席者、填原因備註、看被跳過的人與原因。
  • 警告：手動選到不合適的人會出現紅字提醒（非本班任課、此人今日請假、與本批其他班撞同一人、與今天已排的某班撞同一人）。
  • 確認並記錄：寫入記錄。若會造成某人同一天代理兩班，會先逐條詢問，確認才寫入。旁有列印、重排。

三、老師與班級（管理者設定）

班級

  • 選年級（七／八／九年級／未分年級）與名稱後新增；每年班級數不同沒關係，隨時可增刪。
  • 依班名自動分年級：班名開頭是數字時（例如 701、803、901），按一下就依開頭數字分好——7→七、8→八、9→九年級；判斷不出來的（例如「一班」）維持原樣。套用前會先告訴你要調整幾班、哪些無法判斷，確認才會動。適合開學一次把整批班級分好。
  • 清單依年級分組、可折疊；可直接改名、刪除（刪除後舊記錄仍保留，只顯示為已刪除）；並列出各班目前的任課老師。

老師與序號

  • 輸入姓名與序號新增。序號重複會跳紅字警告（兩人同號輪替會亂）。
  • 每列可改序號、改姓名、勾選任課班級（依年級分行顯示）。
  • 比照最低：學期中才加入的老師，先勾好任課班級再按此鍵，起始次數會對齊各班目前最低，避免他一直被優先排到。

批次匯入老師

  • 貼上名單，一行一位，格式：姓名, 序號, 班級 班級（姓名／序號／班級用逗號或 Tab 分隔；多個班級用空格或逗號）。例：王小明, 1, 701 702。
  • 按預覽會先列出解析結果，標出「資料不全」與「將自動建立的班級」；確認才寫入。找不到的班會自動建立（年級待補，可到班級清單補上）。

四、請假

  • 選老師、日期、理由（選填）後登記。
  • 登記後，該老師當天會被自動跳過，但不算他輪過，下次仍優先輪到。
  • 清單可刪除（例如請假取消）。

五、記錄統計

  • 復原上一次寫入：剛按錯確認時，一鍵撤回上一批。
  • 每位老師輪替狀況：老師人數、班級數、累計代理次數，以及每人的任課班級、累計代理、最近一次日期。設過分界日會註明「自某日起算」。
  • 各班輪替序列：依年級分組；每班顯示「第幾輪．本輪還有幾人未代理」，並用綠色標出下一位優先者，每人附目前次數。
  • 當月統計：選月份，列出每位老師次數、每班次數，以及完整明細（日期／班級／老師／類型／缺席備註），可匯出 CSV（Excel 可直接開、不亂碼）。
  • 完整代理記錄：所有記錄，可刪除單筆（會影響往後輪替計算）。

六、備份與重置

備份

下載備份（JSON）：檔名固定 YYYYMMDD_代導記錄.json。匯入備份：從檔案還原。

頁面上方若出現「有變更尚未備份」，按一下即可下載。

新學年

設一個分界日：從那天起輪替次數歸零，但所有舊記錄照樣保留可查、可匯出。可隨時取消分界改回計算全部。

全部清除（危險）

刪掉所有老師、班級、記錄，回到示範資料，無法復原。設計成三重防呆：按兩次確認、再手動輸入「清除」二字才會執行。請務必先下載備份。

七、資料存在哪、幾個人用

  • 資料只存在這台電腦的這個瀏覽器（不會上傳雲端）。
  • 換電腦、換人、換瀏覽器，或清了快取，資料就不在了——所以務必養成下載備份的習慣，要換機時把 JSON 帶過去用「匯入」還原。
  • 多人／多台要用同一份資料，目前靠匯出／匯入那份 JSON 傳遞。

八、常見狀況怎麼辦

狀況做法
老師臨時說要請假直接在「排代理」點該班排一次即可；不需要先去「請假」登記。（「請假」是要讓某人某天整天都被跳過時才用。）
同一天陸續有好幾班請假每次來就處理當下那一班，系統會自動避開今天已排走的人。
想先看看今天會輪到誰勾「只試排，先不記錄」再按排代理。
排完發現老師沒請假了到「排代理」綠框那筆按 ×，或到「記錄統計」刪掉該筆。
剛按錯確認「記錄統計」按「復原上一次寫入」。
換了新電腦舊電腦「下載備份」→ 新電腦開工具 →「備份與重置」→「匯入備份」。
